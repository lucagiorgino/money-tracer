# money-tracer

This will be a web application for tracking expenses, income, and personal assets. Although it is being developed primarily for personal use, I am designing it as a multi-user application as an exercise to learn and apply new concepts. The goal is to follow best practices in architecture, security, and scalability, treating it as a production-ready system rather than a simple personal tool.
